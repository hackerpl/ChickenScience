## 产品概述

### 产品名称
**小鸡科学**

### 核心目标
解决孩子不学科学的问题，通过科学的**重要性、有趣性、趣味性、成就感、实用性**五个维度，让孩子愿意主动学习科学。

### 产品愿景
让孩子爱上科学，从"不想学"变成"主动学"，培养科学思维和探索精神。

---

## 目标用户

### 主要用户
| 用户类型 | 年龄范围 | 特征 | 需求 |
|---------|----------|------|------|
| **学生** | 小学 - 大学 | 对科学不感兴趣，想法抽象，需要引导 | 游戏化学习，有趣的内容，即时反馈 |
| **家长** | 25-45岁 | 关注孩子学习，希望孩子掌握科学知识 | 监督学习，了解进度，控制内容 |

### 用户画像

#### 学生画像
- **姓名**: 小明
- **年龄**: 10岁
- **年级**: 小学四年级
- **特征**: 对科学课没兴趣，觉得科学无聊，喜欢玩游戏
- **痛点**: 不知道科学的重要性，觉得科学离生活很远
- **期望**: 有趣的学习方式，能和其他小朋友一起玩

#### 家长画像
- **姓名**: 张妈妈
- **年龄**: 35岁
- **职业**: 公司职员
- **特征**: 工作忙，没时间辅导孩子，希望孩子主动学习
- **痛点**: 不知道孩子学了什么，担心孩子沉迷游戏
- **期望**: 能看到孩子的学习情况，控制学习内容

## 产品定位

### 市场定位
- **产品类型**: 教育游戏化小程序

## 功能需求详细说明

### 1. 启动流程

#### 1.1 身份选择

**功能描述**:
用户首次打开小程序时，询问用户身份，提供不同的引导路径。

**流程**:
```
打开小程序 → 显示身份选择页 → 用户选择身份
```

**界面元素**:
- 标题: "你是科学大师还是小白？"
- 选项1: "我是科学大师"（按钮，跳过引导）
- 选项2: "我是小白"（按钮，进入新手指引）

**逻辑规则**:
- 首次使用必须选择身份
- 选择后记录到本地存储
- 以后直接进入主界面（可在"我的"页面重新查看引导）

#### 1.2 新手指引（小白模式）

**功能描述**:
为小白用户提供科学基础知识的文字讲解，帮助快速入门。

**内容形式**:
- 文字讲解（非视频）
- 科学基础知识
- 可点击"下一页"

**页面结构**:
```
━━━━━━━━━━━━━━━━━━
    科学基础知识
━━━━━━━━━━━━━━━━━━
    [文字内容区域]
    可滚动，多页展示

━━━━━━━━━━━━━━━━━━
    [上一页]  [下一页]
━━━━━━━━━━━━━━━━━━
```

**防作弊机制**:
- 翻页速度检测（如：小于3秒翻页视为快速翻页）
- 快速翻页的引导归档到左下角
- 以后可在"我的"页面重新查看

**存储规则**:
- 记录用户是否认真阅读
- 未认真阅读的引导归档存储
- 可随时重新查看

### 2. 学习系统

#### 2.1 答题系统

**功能描述**:
核心学习功能，通过答题学习科学知识。

**题型设计**:

| 题型 | 说明 | 防作弊机制 |
|------|------|-------------|
| **判断题** | 题目 + 选项（对/错） | 半对半错陷阱选项 |
| **填空题** | 题目 + 输入框 | - |
| **挑战题** | 更高难度题目 | 需点击"挑战模式"解锁 |

**答题流程**:
```
显示题目 → 用户答题 → 系统判断
    ├─ 正确 → 下一题
    └─ 错误 → 提示错误，允许重试
        ├─ 重试1次正确 → 下一题
        ├─ 重试2次正确 → 下一题
        └─ 重试3次均错误 → 强制观看讲解视频 → 继续答题
```

**题目展示**:
```
━━━━━━━━━━━━━━━━━━
  第 5 题 / 共 10 题
━━━━━━━━━━━━━━━━━━
  题目内容:
  水星上有水吗？

  A. 有
  B. 没有

━━━━━━━━━━━━━━━━━━
  [A]  [B]
━━━━━━━━━━━━━━━━━━
```

**挑战模式**:
- 点击"挑战模式"按钮解锁
- 题目难度高于当前年级
- 答对获得更多金币/积分

#### 2.2 视频讲解

**功能描述**:
用户答错3次后，强制观看讲解视频。

**视频来源**:
- 外链优质教育视频（B站、腾讯视频等）
- 复用现有教育资源

**强制观看机制**:
- 视频全屏播放
- 禁止滑出（需视频结束才能关闭）
- 视频结束后显示"继续答题"按钮

**视频展示**:
```
━━━━━━━━━━━━━━━━━━
  [全屏视频播放器]
━━━━━━━━━━━━━━━━━━
  强制观看中...
  00:35 / 03:20
━━━━━━━━━━━━━━━━━━
```

#### 2.3 错题档案

**功能描述**:
记录所有错题，方便用户复习。

**存储结构**:
- 错题ID
- 题目内容
- 用户答案
- 正确答案
- 错误次数
- 错误时间
- 是否已掌握

**功能**:
- 查看错题列表
- 答错题
- 标记为已掌握

#### 2.4 每周复习

**功能描述**:
每周提醒用户复习错题，必须完成才能使用其他功能。

**触发机制**:
- 每周日检查是否有未复习的错题
- 如有错题，弹出复习提醒
- 用户必须完成复习才能继续使用其他功能

**复习流程**:
```
周日登录 → 检查错题 → 有错题？
    ├─ 是 → 进入错题复习模式
    └─ 否 → 正常进入主界面
```

**提醒界面**:
```
━━━━━━━━━━━━━━━━━━
  💡 本周错题提醒
━━━━━━━━━━━━━━━━━━
  本周你有 5 道错题
  需要复习完成后才能继续使用
  其他功能哦～

  [立即复习]
━━━━━━━━━━━━━━━━━━
```

#### 2.5 重点复习库

**功能描述**:
错题再错后进入重点复习库，每周必做，直到不再错。

**触发条件**:
- 错题复习时再次答错
- 自动进入重点复习库

**复习规则**:
- 每周必须完成重点复习
- 直到连续3次正确才移出重点库
- 未完成重点复习无法使用其他功能

**重点复习库界面**:
```
━━━━━━━━━━━━━━━━━━
  🔥 重点复习库
━━━━━━━━━━━━━━━━━━
  本周重点错题: 3 道

  1. 水星上有水吗？
     错误次数: 5
     状态: 未掌握

  2. 光速是多少？
     错误次数: 3
     状态: 未掌握

  [开始复习]
━━━━━━━━━━━━━━━━━━
```

### 3. 对战系统

#### 3.1 双人对战（1v1）

**功能描述**:
附近玩家自动匹配，互相出题对战。

**匹配规则**:
- 地理位置: 附近玩家（500米内）
- 年级匹配: 同一年级或相邻年级
- VIP等级: 同一等级段（如1-5, 6-10, 11-15, 16-20）

**对战流程**:
```
点击对战 → 匹配对手 → 石头剪刀布 → 赢方先出题
→ 双方互相出题（各10道）→ 判定胜负 → 奖励结算
```

**石头剪刀布界面**:
```
━━━━━━━━━━━━━━━━━━
  🎮 对战准备
━━━━━━━━━━━━━━━━━━
  你的对手: 科学达人
  VIP等级: 5

  石头剪刀布！
  赢的人先出题

  [✊]  [✌️]  [✋]
━━━━━━━━━━━━━━━━━━
```

**出题界面**:
```
━━━━━━━━━━━━━━━━━━
  轮到你出题了
━━━━━━━━━━━━━━━━━━
  请出1道科学题:
  （输入题目内容）
  （输入正确答案）

  [出题]  [跳过]
━━━━━━━━━━━━━━━━━━
```

**内容过滤**:
- 系统自动检测题目内容
- 过滤无关话题（如: "你好"、"吃饭了吗"）
- 过滤不良内容（敏感词检测）

**判定规则**:
- 一方答不出对方题目 → 输
- 双方都答出所有题目 → 答对更多题目的一方赢
- 平局 → 石头剪刀布决定

**奖励规则**:
- 赢方: 获得金币（VIP获得2金币，普通用户获得1金币）
- 输方: 无奖励
- 平局: 双方各获得1金币

#### 3.2 多人对战（VIP20）

**功能描述**:
VIP20用户可进行2v2多人对战。

**解锁条件**:
- VIP等级 ≥ 20

**对战模式**:
- 每方2人（共4人）
- 可选择出题方式:
  - **我方出题**: 我方给对方出题
  - **对方出题**: 对方给我方出题

**匹配规则**:
- 自动匹配3名同等级VIP用户
- 4人进入对战房间

**对战界面**:
```
━━━━━━━━━━━━━━━━━━
  🎮 多人对战（2v2）
━━━━━━━━━━━━━━━━━━
  我方 (蓝色队):
  👤 你
  👤 队友: 科学小达人

  对方 (红色队):
  👤 对手1: 知识大王
  👤 对手2: 答题高手

  当前轮次: 3 / 10
━━━━━━━━━━━━━━━━━━
```

#### 4.2 私聊功能

**功能描述**:
好友间可进行私聊交流。

**聊天入口**:
- 必须在"我的"页面进入好友聊天

**聊天界面**:
```
━━━━━━━━━━━━━━━━━━
  💬 科学达人
━━━━━━━━━━━━━━━━━━
  你: 你觉得这道题难吗？
  科学达人: 还可以，就是要记
  住很多公式。

  [输入框]
━━━━━━━━━━━━━━━━━━
```

**内容过滤**:
- 自动过滤无关话题
- 自动过滤不良内容
- 仅允许与学科相关的交流

#### 4.3 内容过滤规则

**过滤内容**:
- 无关话题: "你好"、"在吗"、"吃饭了吗"
- 不良内容: 敏感词、广告、骚扰信息
- 非学科内容: 娱乐八卦、游戏讨论

**过滤机制**:
- 发送前检测内容
- 发现违规内容，提示"内容不符合规则"
- 记录被过滤内容，家长可查看

### 5. 激励系统

#### 5.1 排行榜

**功能描述**:
展示用户排名，激发竞争欲。

**排行榜类型**:

| 排行榜 | 说明 | 更新频率 |
|--------|------|----------|
| **单人榜** | 根据答题数量/正确率排名 | 每日更新 |
| **对战榜** | 根据对战胜利次数排名 | 每周更新 |

**排行榜界面**:
```
━━━━━━━━━━━━━━━━━━
  🏆 单人排行榜
━━━━━━━━━━━━━━━━━━
  1️⃣ 科学达人    1500题  98%
  2️⃣ 知识大王    1400题  97%
  3️⃣ 答题高手    1300题  96%
  4️⃣ 小明（我）   1200题  95%
  5️⃣ 科学小天才   1100题  94%

  [对战榜]  [我的排名]
━━━━━━━━━━━━━━━━━━
```

**VIP晋升规则**:
- 排行榜前5名 → 升1级VIP
- 每周更新一次
- 年度重置（春节）

#### 5.2 VIP系统

**功能描述**:
VIP等级体系，解锁更多功能。

**VIP等级与特权**:

| VIP等级 | 解锁功能 | 描述 |
|---------|----------|------|

## 技术方案

### 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端 | 微信小程序 | WXML + WXSS + JavaScript |
| 后端 | 云开发（腾讯云） | Serverless架构 |
| 数据库 | 云数据库（MongoDB） | 文档型，灵活存储 |
| 实时通讯 | WebSocket | 支持对战、私聊 |
| AI接口 | 智谱AI / 通义千问 | 国内API，稳定可用 |
| 视频 | 腾讯视频云 / 外链 | 复用现有教育资源 |
| 内容过滤 | 敏感词库 + AI审核 | 过滤无关话题和不良内容 |

### 架构设计

```

│  └──────────┘ └──────────┘ └──────────┘ │
└─────────────────────────────────────────┘
                    ↕
┌─────────────────────────────────────────┐
│            云开发后端服务                │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐ │
│  │ 云函数   │ │ 云数据库 │ │ 云存储   │ │
│  └──────────┘ └──────────┘ └──────────┘ │
└─────────────────────────────────────────┘
                    ↕
┌─────────────────────────────────────────┐
│           第三方服务                    │

│  └──────────┘ └──────────┘ └──────────┘ │
```

### 核心技术难点
